Remote System Explorer Model

The underlying containment model for the RSE artifacts is shown here.

  • Profiles are managed by a profile manager, and contain:
    • Connection Pools, one per profile, which contain
      • Connections
    • Filter pools, keyed by Subsystem Factory, and managed by a filter pool manager which contain
      • Filters, which contain
        • Filter strings
    • User actions, keyed by Subsystem Factory, which contain
      • References to named types
    • Compile commands
  • Subsystem Factories contain:
    • Subsystems, keyed by connection, which contain
      • References to filter pools, which are managed by filter pool managers
    • Named types
